# DESIGN

## Overview

Premium minimalist landing for an enterprise SaaS. Atmosphere — Density 2 / Variance 3 / Motion 3.

## Colors

Two-tone neutral foundation with a single accent. Primary `#0A0A0A` for type, background `#FAFAFA` for surfaces, accent `#FF4D2D` reserved for CTAs.

## Typography

Söhne Breit display for the hero, Söhne for body, Söhne Mono for code. Type scale `1.125` → `1.5` → `2.25` → `4.5rem`.

## Layout

12-column grid, max-width 1280px, generous whitespace at section boundaries (≥ 8rem on desktop).

## Elevation & Depth

No shadows. Depth comes from typography weight contrast and accent color saturation.

## Shapes

Hard edges by default. Cards use `rounded.md`, buttons use `rounded.md`. No `rounded.full` except for avatars.

## Components

Button (primary / secondary), Card (interactive / static), Hero (full-bleed / contained).

## Do's and Don'ts

- Do — let typography carry hierarchy
- Do — reserve the accent for one CTA per viewport
- Don't — add gradients
- Don't — animate text without reason
